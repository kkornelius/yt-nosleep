# YT NoSleep 🌙

Ekstensi browser untuk YouTube yang secara otomatis mengklik tombol "Continue watching" dan melakukan auto-skip iklan.

## Fitur

- **NonStop (Lanjutkan Menonton)**: Secara otomatis mengklik tombol "Continue watching" saat muncul dialog konfirmasi di YouTube, sehingga video dapat berlanjut tanpa interupsi.
- **Auto-Skip Iklan**: Mendeteksi dan secara otomatis melewati iklan yang dapat di-skip di YouTube.

## Cara Instalasi

### Mode Developer (Chrome/Edge/Brave)

1. Clone atau download repository ini
2. Buka browser dan navigasi ke halaman ekstensi:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Brave: `brave://extensions/`
3. Aktifkan **Mode Developer** (toggle di pojok kanan atas)
4. Klik **Load unpacked** (Muat yang belum dikemas)
5. Pilih folder `yt-nosleep` yang berisi file `manifest.json`
6. Ekstensi siap digunakan!
